# Menubar

Menubar is the library which is used to pass props like data(Menu-items) , background-color , text-color , amimations etc...

## Import 

Menubar is exported as default from 'cb-react-menubar' . You can use or assign any other name if you want while importing.

```
import Menubar from 'cb-react-menubar'

```

## Usage

You can use it anywhere you want

```
const App=()=>{

  return (
    <div>
     <Menubar data={MenuItems}>
    </div>
  )
}
```
