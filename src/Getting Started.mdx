# Getting Started

## Installation

Start by installing `react-responsive-multi-level-menu` as a dependency with `npm` :

```
npm i react-responsive-multi-level-menu
```

or if you are using `yarn` :

```
yarn add react-responsive-multi-level-menu
```

## Basic Usage


### 1.Define the data

* Define the data(menu items) as per the given format , which you want to render in the Menubar.

* If you want nested items for a specific item , Give it in items array.

* If you dont want nested items for a specific item , dont declare the items array or give a empty array for the items.

```js
const menuItems = [
  { value: "Fashion", items: [{ value: "Men", items: [{ value: "Shirts" }] }] },
  { value: "Electronics", items: [] },
  { value: "Furnitures", items: [] },
  { value: "Jewelery&watches", items: [] }
];
export default MenuItems;

```


### 2. Use the Menubar component anywhere in your code

Use the Menubar component anywhere and pass in the data prop.

```js

import Menubar from "react-responsive-multi-level-menu";
import MenuItems from "../path/where/menuItems/present";

const app = () => {
  <div>
    <h1>This is Animated Menu-bar</h1>
    <Menubar data={MenuItems} />
  </div>;
};


```

### 3. Styling the Menubar

* You can style the Menubar by passing background-color and text-color.

* You can even have your own className and pass to the Menubar.

```js
import Menubar from "react-responsive-multi-level-menu";
import MenuItems from "../path/where/menuItems/present";
import "./index.css";

const app = () => {
  <div>
    <h1>This is Animated Menu-bar</h1>
    <Menubar
      backgroundColor="blue"
      textColor="white"
      className="menuBar"
      data={MenuItems}
    />
  </div>;
};


```

### 4.Animations

* You are provided with various animation effects along with menubar.

* It should be an array which denotes the entering and exiting of the menuItems in the menuBar.

```js
import Menubar from 'react-responsive-multi-level-menu';
import MenuItems from "../path/where/menuItems/present";
import "./index.css"

const app=()=>{

  const animationNames=['slideIn','slideOut']

  <div>
    <h1>This is Animated Menu-bar</h1>
    <Menubar animation={animationNames} data={MenuItems}/>
  </div>
}

```

### 5.Callback function

* You are provided with a onClick function which you can pass your own function and on clicking on the menu item
  will call that function and specific object is passed to you.

```js
import Menubar from "react-responsive-multi-level-menu";
import MenuItems from "../path/where/menuItems/present";
import "./index.css";

const app = () => {
  const thisIsFunction = item => {
    alert(item.value);
  };

  <div>
    <Menubar data={MenuItems} onClick={thisIsFunction} />
  </div>;
};

```
