# Props


There are many props which you can pass to menubar and make it as you want.

<Props />


## Usage

### example-1

* This is the example for the props - data , backgroundColor , textColor and animation

* data is the required prop to be passed to the menubar.

* data props must be passed in the defined format.


```js
import Menubar from "cb-react-menubar"
const App=()=>{

const menuItems = [
  { value: "Fashion", items: [{ value: "Men", items: [{ value: "Shirts" }] }] },
  { value: "Electronics", items: [] },
  { value: "Furnitures", items: [] },
  { value: "Jewelery&watches", items: [] }
];

 return(
    <div>
      <Menubar data={MenuItems}
       backgroundColor="green" 
       textColor="white" 
       animation=['fadeIn'.'fadeOut'] 
      />
    </div>
 )
}

```

### example-2

* This is the example for props - className , hamBurgerClassName and iconColor 

* To override classname for the menubar . you can write your own css and send the classname

* You can also override the classname of hamBurgerMenu icon


```css

.menubar{
  margin-right:50px;
}

.hamBurgerMenu{
  height:30px;
  width:30px
}


```

```js

import Menubar from "cb-react-menubar";
import "./style.css";
const App = () => {
  const menuItems = [
    { value: "Fashion" },
    { value: "Electronics", items: [] },
    { value: "Furnitures", items: [] },
    { value: "Jewelery&watches", items: [] }
  ];
  return (
    <div>
      <Menubar
        data={menuItems}
        className="menubar"
        hamBurgerClassName="hamBurgerMenu"
        iconColor="black"
      />
    </div>
  );
};


```

### example-3

* This is example for the prop - onClick and menuContainerWidth

```js

import Menubar from "cb-react-menubar";
const App = () => {
  const menuItems = [
    { value: "Fashion" },
    { value: "Electronics", items: [] },
    { value: "Furnitures", items: [] },
    { value: "Jewelery&watches", items: [] }
  ];

  const callThisFunction = item => {
    alert(item.value);
  };

  return (
    <div>
      <Menubar data={menuItems} onClick={callThisFunction} menuContainerWidth={300}/>
    </div>
  );
};

```
